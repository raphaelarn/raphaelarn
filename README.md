## Architecture

<br/>

```
  CLIENT LAYER
  ─────────────────────────────────────────────────
  React.js              Next.js (SSR / SSG / ISR)

          │                        │
          └──────────┬─────────────┘
                     │  REST / GraphQL
                     ▼

  API LAYER
  ─────────────────────────────────────────────────
  NestJS (Modules · Guards · Interceptors)
  Node.js Runtime

          │                        │
          ▼                        ▼

  DATA LAYER
  ──────────────────   ──────────────────────────
  PostgreSQL            MySQL
  (Primary DB)          (Relational DB)

  CI/CD
  ─────────────────────────────────────────────────
  GitLab CI  →  Build  →  Test  →  Deploy
```
